**Topics:** Python · Data Science · Machine Learning · Deep Learning · Computer Vision

## Setup

Install Python dependencies with:

```bash
pip install -r requirements.txt
```

Key libraries used across this repo: `numpy`, `pandas`, `matplotlib`, `scikit-learn`, `tensorflow`, `keras`, `opencv-python`, `jupyter`.
